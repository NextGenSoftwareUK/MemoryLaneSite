# Memory Lane

**Preserve What Matters Most · OASIS Omniverse**

Memory Lane is an OASIS Omniverse app for curating and preserving your most precious memories in a beautiful, interactive timeline. Share with family, relive moments, and earn karma for keeping the past alive and meaningful.

## What is Memory Lane?

- **Interactive memory timeline** — arrange memories chronologically with photos, text, audio and video
- **Family sharing** — invite family members to contribute their own memories to a shared timeline
- **Memory prompts** — guided questions to help capture memories before they fade
- **Legacy profiles** — create a lasting digital legacy for loved ones who have passed
- **Karma integration** — earning karma for uploading, sharing and preserving memories via OASIS Web4 API

## Use Cases

- Family history and genealogy
- Childhood and life milestones
- Travel and adventure memories
- Tributes to loved ones
- Business and project history

## Tech Stack

| Layer | Detail |
|---|---|
| Front-end | Single-file `index.html` — inline CSS + vanilla JS |
| Fonts | Orbitron, Rajdhani, Share Tech Mono (Google Fonts) |
| OASIS API | `@oasisomniverse/web4-api@2.0.2` via esm.sh |
| Hosting | Static — any CDN or static host |

## Running Locally

```bash
npx serve .
# or
python -m http.server 8080
```
